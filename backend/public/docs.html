API documentation

GET/health

POST/early-access

Early access

Submit name, email, company, country, project, model, and usage.

POST/admin/create-key

Create API key

Requires x-admin-secret. The raw key is returned once.

GET/admin/stats · /admin/leads · /admin/logs

Administrator data

Protected overview, lead, and usage data.

POST/v1/chat/completions

Chat completions

Send an OpenAI-compatible non-streaming chat request with a Bearer API key.

from openai import OpenAI

client = OpenAI(
    api_key="cllm_your_key",
    base_url="https://your-service.onrender.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Hello"}]
)

Errors and limits

Errors use an OpenAI-style error object. Streaming is not supported. Early-access submissions are limited to five per 15 minutes.
